# Remote Desktop & remote support access

**Applies to:** staff who need to connect to a remote PC, or receive IT remote support · **KB ID:** KB-0009
**Last reviewed:** 2026-06-21 · **Owner:** Service Desk

## What this covers
How to connect to your work PC remotely, and what to expect when IT helps you via a remote-support session.

## Connect to a remote work PC (Remote Desktop)
1. Make sure you're on the network — if remote, connect the **VPN** first (KB-0003).
2. Open **Remote Desktop Connection** (search "Remote Desktop" in Start), or the tool IT provided.
3. Enter the **PC name** IT gave you (e.g. **`DESK-1102`**) and sign in with your work account + **MFA**.
4. You'll see the remote desktop. **Sign out** (don't just close) when you're done.

> Access to Remote Desktop must be **granted** for your account/PC — if you can't connect, request it via
> the Service Desk (with manager approval).

## When IT helps you remotely (Quick Assist / support tool)
- IT will start a session **only after verifying who you are** and **with your consent** — you'll approve a
  prompt or share a code **they** give you.
- **Watch what's happening** on your screen during the session.
- **Security:** IT will **never** call you unprompted asking you to install remote-access software or read
  out a code from a random pop-up. If someone does, **hang up and report it** — that's a scam.

## Still not working?
- Can't reach the remote PC → confirm VPN is connected (KB-0003) and you have the right PC name; contact the
  desk. Reference **KB-0009**.
