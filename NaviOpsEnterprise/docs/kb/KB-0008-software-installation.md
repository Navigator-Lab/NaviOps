# Request & install software

**Applies to:** staff who need an application installed · **KB ID:** KB-0008
**Last reviewed:** 2026-06-21 · **Owner:** Service Desk

## What this covers
How to get approved software installed yourself, and how to request software that isn't in the catalog.

## Standard (approved) software — self-service
1. Open the **Company Portal** (or Software Center) from the Start menu.
2. Find the app and click **Install** — it deploys without needing admin rights.
3. Give it a few minutes; the app appears in your Start menu.

## Software that isn't in the catalog
Don't download installers from the web — they may be **unsafe** or **unlicensed**. Instead:
1. Submit a request to the Service Desk with: the **app name**, **what you need it for**, and any
   **license** info.
2. IT reviews it for **security, licensing, and compatibility**, then installs an approved version (or
   suggests an approved equivalent).
3. Urgent? Tell the desk — there's often an **already-approved** tool that does the same thing today.

## Removing software
Use **Settings → Apps → Installed apps** to uninstall apps you no longer need, or ask the desk if it won't
remove cleanly.

## Still not working?
- An install fails or says you need admin rights → contact the desk (don't try to bypass it). Reference
  **KB-0008** with the **app name** and the **exact error**.
