# Runbooks — what you produce here

**This folder starts empty on purpose — you fill it as you complete lessons.** A runbook is a
technician-facing, repeatable procedure: *"when this ticket/condition occurs, do exactly this."*

## How to create one
1. Copy `docs/templates/runbook.md`.
2. Fill it in for the lesson's task (ordered steps, a verify at each gate, rollback, escalate).
3. Save it here as `NN-topic.md` (or a clear name), sanitized (placeholders only — `navi.project.md` HR#1).
