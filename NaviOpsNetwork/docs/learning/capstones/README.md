# Capstones — NaviOpsNetwork

Three end-to-end projects that integrate the curriculum and become the **portfolio
centerpieces** (`alignment/PORTFOLIO-STRATEGY.md`). Each is a Lesson (34/35/36) but larger:
it produces a multi-artifact deliverable + a `PORTFOLIO.md` (resume bullets + interview
talking points).

| # | Capstone | Proves | Primary role |
|---|---|---|---|
| 34 | [CCNA Capstone](34-ccna-capstone.md) | design + build + verify a routed/switched network with services | Junior Network Engineer |
| 35 | [NOC Capstone](35-noc-capstone.md) | run a simulated 24/7 NOC shift end-to-end | NOC Technician / NetOps |
| 36 | [Network-Security Capstone](36-network-security-capstone.md) | detect + respond to an attack with IDS + SIEM | SOC / Security Analyst (net) |

**Prerequisites:** complete the relevant module lessons first (see each plan). Each capstone
uses lab/RFC-1918 ranges and tools the operator owns — no third-party targets.
